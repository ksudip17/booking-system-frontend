# BookGarnus Frontend

Modern, responsive frontend for BookGarnus booking management system.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- ES6+ JavaScript

## Features
- Clean, modern UI
- Responsive design
- Real-time CRUD operations
- Form validation
- Loading states
- Error handling

## Setup

1. Clone the repository
2. Install dependencies:
```bash
   npm install
```
3. Run development server:
```bash
   npm run dev
```
4. Build for production:
```bash
   npm run build
```

## Project Structure
```
├── src/
│   ├── components/      # Reusable components
│   ├── services/        # API service layer
│   ├── utils/           # Utility functions
│   ├── App.jsx          # Main app component
│   └── main.jsx         # Entry point
```
